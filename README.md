## Spring boot blog 


### Quick Start

- JDK 1.8 (or Java 10)
- Maven 3.5
- Git

```
git clone https://github.com/rslozl/Spring-Boot-E-Commerce
cd Spring-Boot-E-Commerce
mvn Spring-Boot-E-Commerce:run
```


### Project Convention

#### Package Structure

```
com.millky.blog
    └── application
        └── aop
        └── configuration
        └── utility
    └── domain
        └── model
            └── command
            └── entity
            └── exception
        └── repository
        └── service
    └── infrastructure
        └── dao
    └── presentation
        └── controller
            └── rest
            └── web
        └── support
            └── result
```

#### Environment
- Java version: 8 Update 172 or 10.0.1
- Spring Boot version: 2.0.2
- Maven version: 3.5.2
- Default Encoding: UTF-8
- Default SCM : git


### Run 
- windows : http://192.168.99.100:8080
- mac os : http://localhost:8080
